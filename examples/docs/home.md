## 项目开始
---

项目的初衷是封装一个基于Vue的UI组件库，但由于技术水平不够，现在最主要的目的是想边学习边封装，顺便记录每个步骤，以及在此过程中遇到的难点及体会。
<br>
参考文章：[从零开始搭建Vue组件库 VV-UI](https://zhuanlan.zhihu.com/p/30948290)

### 项目搭建
使用 vue cli 3.0.0版本以上，在node或cmd中输入以下命令创建项目
```js
vue create project-name
```
